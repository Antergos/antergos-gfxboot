Opções de Boot

Existem dois tipos de opções de boot disponíveis. Primeiro, há opções que afetam o instalador. Segundo, há opções do kernel. Algumas das opções mais comuns são:

a) opções do instalador

  • Instalar -- seleciona uma fonte de instalação
  • opções de rede -- as opções de rede
  • Opções do VNC -- opções para instalação via VNC

b) opções do kernel

  • Splash -- influencia o comportamento da splash screen
  • apm -- alterna o gerenciamento de energia
  • ACPI -- ACPI (Advanced Configuration and Power Interface)
  • ide -- controla o subsistema IDE
