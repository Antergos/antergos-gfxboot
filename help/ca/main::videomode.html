Selecció del mode de vídeo

Premeu @@@fkey_video@@@ per a obtenir una llista dels modes de vídeo implementats per la vostra targeta gràfica. Se selecciona automàticament el mode més alt que pot acceptar la vostra pantalla.

Possiblement el vostre monitor no pugui detectar-se automàticament, En aquest cas, seleccioneu el vostre mode preferit manualment.

Si teniu problemes amb la targeta gràfica en iniciar, proveu el mode text com a possible solució.
