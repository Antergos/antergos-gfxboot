Ajuda del gestor d'arrencada

Benvingut a @@@product@@@

Useu aquest menú per a seleccionar la funció desitjada. Si teniu problemes per a navegar en aquest sistema d'ajuda, premeu @@@fkey_help@@@ per a entrar a la descripció del sistema d'ajuda. Les funcions principals d'aquest menú són:

Inicia el sistema live: Seleccioneu aquesta opció per començar a usar el sistema @@@product@@@ live.

Inicia (controladors no lliures): Seleccioneu aquesta opció per començar a usar el sistema @@@product@@@ live, incloent controladors no lliures.

Inicia des de disc dur: Aquesta selecció no afecta el sistema instal·lat. Només inicia un sistema operatiu prèviament instal·lat.

Eina de detecció de maquinari: Aquesta selecció obre una eina per a comprovar el vostre maquinari. Aquesta eina no afecta o determina el maquinari usat per a instal·lar @@@product@@@.

Deshabilita ACPI: Molts dels ordinador venuts avui dia tenen una implementació ACPI incompleta o incorrecta. Aquesta selecció deshabilita la implementació ACPI al nucli del sistema, però conserva moltes característiques de rendiment, com ara DMA per a discs durs IDE.

Deshabilita APIC local: Si la instal·lació normal falla, pot ser que el maquinari del sistema no implementi APIC local. Si aquest sembla ser el cas. useu aquesta opció per a instal·lar sense implementació APIC local.

Prova de memòria: La prova de memòria és útil per a més coses a part de comprovar la instal·lació de nous mòduls de memòria. És una prova d'estres d'una gran part del vostre ordinador i pot detectar problemes de maquinari.

Opcions d'arrencada: Les opcions d'arrencada poden canviar completament el comportament del vostre sistema. Són arranjaments del nucli del sistema.

@@@fkey_help@@@ Ajuda: És sensible al context. Mostra pantalles diferents depenent de l'element actiu de la pantalla d'inici. També hi ha disponible una descripció d'aquest sistema d'ajuda.

@@@fkey_lang@@@: Estableix l'idioma i el mapa de teclat usats pel carregador d'arrencada. Aquesta opció determina l'arranjament del sistema live.

@@@fkey_video@@@ Mode de vídeo: Aquí podeu triar entre diferents resolucions de pantalla pel sistema live. Si trobeu problemes amb la instal·lació gràfica, el Mode text podria ser una solució.
