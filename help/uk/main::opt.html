Параметри завантаження

Є два типи наявних параметрів завантаження. По-перше, є параметри, які впливають на програму встановлення. По-друге, є параметри ядра. Найпоширеніші параметри:

a) параметри встановлення

  • встановити — виберіть джерело встановлення
  • мережні параметри — мережні параметри
  • параметри vnc — параметри для встановлення через VNC

b) параметри ядра

  • заставка — керування поведінкою заставки
  • apm — перемикання керування живленням
  • acpi — інтерфейс розширеного налаштування та керування живленням
  • ide — керування підсистемою IDE
