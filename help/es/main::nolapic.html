Instalación: APIC local inhabilitado

Normalmente no se producen problemas con el sistema APIC (controlador avanzado de interrupciones programables, del inglés Advanced Programmable Interrupt Controller). Pero, si se produce alguno, pruebe esta opción para arrancar sin compatibilidad local con APIC.

No confundir con ACPI (interfaz avanzada de configuración y alimentación, del inglés Advanced Configuration and Power Interface).
