Ayuda del cargador de arranque

Le damos la bienvenida a @@@product@@@

Utilice este menú para seleccionar la función deseada. Si tiene problemas para desplazarse por este sistema de ayuda, pulse @@@fkey_help@@@ para acceder al Descripción del sistema de ayuda. Las funciones principales de este menú son:

Arrancar desde el disco duro: la selección de esta opción no hará nada en el sistema. Sólo iniciará un sistema operativo instalado previamente.

Instalación: este modo de instalación funciona en la mayoría de los equipos. Si experimenta un bloqueo del sistema durante el arranque o problemas a la hora de detectar los componentes de hardware, como los controladores de disco o las tarjetas de red, pruebe una de las siguientes opciones de instalación.

Instalación: ACPI inhabilitado: muchos de los equipos que se venden actualmente cuentan con implementaciones de ACPI incompletas o erróneas. Esta opción inhabilita la compatibilidad con ACPI del núcleo, pero mantiene habilitadas muchas de las funciones de rendimiento, como DMA para los discos duros IDE.

Instalación: APIC local inhabilitado: si la instalación normal no se lleva a cabo correctamente, puede deberse a que el hardware del sistema no sea compatible con APIC local. Si cree que es el caso, utilice esta opción para instalar sin compatibilidad con APIC local.

Instalación: valores de configuración seguros: si no ha tenido éxito con Instalación, puede que al seleccionar esta opción se resuelva el problema.

Sistema de rescate: esta imagen de arranque inicia un pequeño sistema Linux en la memoria RAM, lo que resulta útil si el sistema no se inicia correctamente. Tras arrancar este sistema, se debe entrar como usuario Root.

Prueba de firmware: ejecutar una herramienta de prueba del BIOS que valida el sistema ACPI y otras partes del BIOS.

Prueba de memoria: la prueba de la memoria resulta útil no sólo para comprobar la instalación de nuevos módulos de memoria. Se trata de una prueba de carga de gran parte del sistema y puede poner de manifiesto otros problemas de hardware.

Opciones de arranque: las opciones de arranque pueden cambiar por completo el comportamiento del sistema. Son valores para el núcleo.

@@@fkey_help@@@ Ayuda: es sensible al contexto. Muestra distintas pantallas según el elemento activo de la pantalla de arranque. También hay disponible una descripción de este sistema de ayuda.

@@@fkey_lang@@@: definir el idioma y la distribución de teclado del cargador de arranque.

@@@fkey_video@@@ Modo de vídeo: aquí debe elegir entre varias resoluciones de pantalla durante la instalación. Si se producen problemas con la instalación gráfica, puede que la opción Modo de texto sirva de solución.

@@@fkey_inst@@@ Origen de instalación: seleccionar el origen de instalación.

@@@fkey_dud@@@ Actualización del controlador: en los equipos muy nuevos, puede que sea necesario actualizar los controladores para instalar el sistema.
