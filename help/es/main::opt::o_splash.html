Opciones del núcleo: splash

La pantalla inicial es la imagen que se muestra durante el inicio del sistema.

  • splash=0 -- La pantalla inicial se desactiva. Esto puede resultar útil con monitores muy antiguos o si se producen errores.
  • splash=verbose -- Activa la pantalla inicial. Los mensajes de arranque y del núcleo se siguen mostrando.
  • splash=silent -- Activa la pantalla inicial. En lugar de mensajes se muestra una barra de progreso.
