Možnosti instalátoru: možnosti sítě

Nyní je možné nakonfigurovat síťové rozhraní. Hardware bude vyhledán později pomocí modulu YaST2. Pro konfiguraci síťové karty je nutné zadat minimálně adresu IP hostitele a síťovou masku. Příklad:

  • hostip=192.168.0.10 netmask=255.255.255.0

nebo v kratší podobě:

  • hostip=192.168.0.10/24

Pokud jste zadali síťová instalace a nezadáte oba tyto parametry, pokusí se instalační program nakonfigurovat síťové rozhraní pomocí protokolu dhcp. Potřebujete-li výchozí bránu, zadejte ji pomocí parametru Brána. Například:

  • gateway=192.168.0.8
