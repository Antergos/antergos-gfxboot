ਹਾਰਡ ਡਿਸਕ ਤੋਂ ਬੂਟ

Select ਇੰਸਟਾਲ ਹੋਇਆ OS ਬੂਟ ਕਰੋ to start the system installed on your local hard disk. This system must be installed properly, because only the MBR (Master Boot Record) on the first hard disk is started. The device ID of the first hard disk is provided by the BIOS of the computer.

ਇਹ ਚੋਣ ਕਰੋ, ਜੇ ਤੁਸੀਂ ਆਪਣੀ ਡਰਾਇਵ ਵਿੱਚੋਂ CD ਜਾਂ DVD ਨੂੰ ਹਟਾਉਣਾ ਭੁੱਲ ਗਏ ਅਤੇ ਆਪਣੇ ਕੰਪਿਊਟਰ ਨੂੰ ਹਾਰਡ ਡਿਸਕ ਤੋਂ ਬੂਟ ਕਰਵਾਉਣਾ ਚਾਹੁੰਦੇ ਹੋ।
