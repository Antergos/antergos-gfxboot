Start fra harddisk

VelgStart installert OS for å starte systemet som er installert på den lokale harddisken. Dette systemet må være riktig installert fordi bare MBR (Master Boot Record) på første harddisk vil bli startet. Enhets-ID-en for første harddisk hentes fra datamaskinens BIOS.

Bruk dette valget hvis du glemte å fjerne CD-en eller DVD-en fra stasjonen og vil starte datamaskinen fra harddisken.
