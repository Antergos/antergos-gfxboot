Val av videoläge

Klicka på @@@fkey_video@@@ om du vill visa en lista över vilka grafiklägen som stöds av grafikkortet. Det högsta läget som kan visas på bildskärmen är förvalt.

Bildskärmen kanske inte kan identifieras automatiskt. I så fall väljer du läge manuellt.

Om det uppstår problem med grafikkortet under installationen kan du komma runt problemet med hjälp av Textläge.
