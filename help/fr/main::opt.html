Options de démarrage

Deux types d'options d'amorçage sont disponibles. En premier lieu, il existe des options qui affectent le programme d'installation. Ensuite, il existe des options du kernel. Certaines des options les plus courantes sont :

a) options du programme d'installation

  • install -- sélectionner une source d'installation
  • network options -- les options réseau
  • vnc options -- options d'installation via VNC

b) options du kernel

  • splash -- déterminer le comportement de l'écran de démarrage
  • apm -- activer/désactiver la fonction de gestion de l'alimentation
  • ACPI -- interface de gestion de l'alimentation et de configuration avancée
  • ide -- contrôler le sous-système IDE
