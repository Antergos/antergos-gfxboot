Installation -- Paramètres sécurisés

Sélectionnez Installation -- Paramètres sécurisés si votre système reste bloqué lors de l'installation ou si des erreurs non reproductibles se produisent. Cette option désactive DMA pour les unités IDE et toutes les fonctions de gestion de l'alimentation. Reportez-vous également aux options du kernel de apm, ACPI et ide.
