Opcións do Instalador: install

Por defecto búscase a fonte de instalación nos CD-ROMs locais. Para unha instalación por rede seleccione a opción install. Os protocolos de instalación posibles son

  • FTP
  • NFS
  • HTTP

A sintaxe a usar é coma a dos URLs estándar. Por exemplo, se o servidor se atopa en 192.168.0.1 e desexa facer unha instalación por NFS dende o directorio /install dese servidor, especifique a fonte así:

  • install=nfs://192.168.0.1/install

A tarxeta de rede pode configurarse con dhcp ou pode especificar os parámetros vostede mesmo tal coma se describe en opcións de rede.
