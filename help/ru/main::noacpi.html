Установка - ACPI выключено

Для управления обработкой прерываний современному оборудованию обычно требуется ACPI (усовершенствованный интерфейс для управления конфигурацией и электропитанием). ACPI полностью заменяет прежнюю систему APM.

Если имеются проблемы при загрузке ядра, выберите Установка - ACPI выключено. Известные проблемы с компьютерами, на которых имеются проблемы с ACPI:

  • ядро не отвечает при загрузке
  • Карты PCI не обнаружены или не инициализированы надлежащим образом

Можно попробовать использовать параметр загрузки Тест микропрограмм и посмотреть результаты проверки ACPI.
