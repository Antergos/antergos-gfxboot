Установка - локальный APIC выключен

APIC (усовершенствованный программируемый контроллер прерываний) обычно не вызывает проблем. Но если таковые имеются, попробуйте применить этот параметр для загрузки без локальной поддержки APIC.

Не путайте с ACPI (усовершенствованным интерфейсом для управления конфигурацией и электропитанием).
