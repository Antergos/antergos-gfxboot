Выбор видеорежима

Чтобы получить список видеорежимов, поддерживаемых видеокартой, нажмите @@@fkey_video@@@. По умолчанию будет выделен режим с максимальным разрешением для используемого монитора.

Если монитор не удается обнаружить автоматически, выберите вручную предпочитаемый режим.

Если во время установки в системе возникли проблемы с видеокартой, Текстовый режим может оказать помощь в их устранении.
