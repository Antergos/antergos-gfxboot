Параметры установки: параметры сети

Можно настроить сетевой интерфейс прямо сейчас. YaST2 выполнит обнаружение оборудования позднее. Минимальный набор параметров для настройки сетевой карты включает IP-адрес хоста и сетевую маску. Например:

  • hostip=192.168.0.10 netmask=255.255.255.0

или в краткой форме:

  • hostip=192.168.0.10/24

Если указать сетевая установка, но не указывать оба эти параметра, программа установки попытается настроить сетевой интерфейс с dhcp. Если необходим шлюз по умолчанию, укажите его с помощью параметра Шлюз. Например:

  • gateway=192.168.0.8
