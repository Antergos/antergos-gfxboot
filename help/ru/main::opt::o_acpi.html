Параметры ядра: acpi

ACPI (усовершенствованный интерфейс для управления конфигурацией и электропитанием) - это стандарт, определяющий интерфейс между операционной системой и BIOS для управления электропитанием и конфигурацией. По умолчанию ACPI включается, если определено, что версия BIOS выпущена позднее 2000 года. Имеется несколько общих параметров для управления режимом работы ACPI:

  • pci=noacpi - не использовать ACPI для маршрутизации прерываний PCI
  • acpi=oldboot - активными остаются только компоненты ACPI, связанные с загрузкой
  • acpi=off - полное отключение ACPI
  • acpi=force - включение ACPI, даже если BIOS выпущен ранее 2000 года

Заменяет прежнюю систему apm, особенно на новых компьютерах.
