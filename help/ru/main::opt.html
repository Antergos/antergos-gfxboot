Параметры загрузки

Имеются параметры загрузки двух типов: параметры программы установки и параметры ядра. Вот некоторые из часто используемых параметров:

а) параметры программы установки

  • Установить - выбор источника установки
  • параметры сети - параметры сети
  • Параметры VNC - параметры для установки через VNC

б) параметры ядра

  • Заставка - изменение режима работы заставки
  • apm - переключение управления питанием
  • ACPI - улучшенный интерфейс для управления конфигурацией и электропитанием
  • ide - управление подсистемой IDE
