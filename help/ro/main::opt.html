Opţiuni de boot

Există două tipuri de opţiuni de boot disponibile: opţiunile ce afectează instalatorul şi opţiunile kernel. Câteva dintre opţiunile uzuale sunt:

a) opţiuni instalator

  • instalează -- selectaţi o sursă de instalare
  • opţiuni reţea -- opţiunile reţea
  • opţiuni vnc -- opţiuni pentru instalarea via VNC

b) opţiuni kernel

  • splash -- influenţează comportamentul ecranului de întâmpinare
  • apm -- comută gestiunea alimentării
  • acpi -- configurare avansată şi interfaţă alimentare
  • ide -- controlează subsistemul IDE
