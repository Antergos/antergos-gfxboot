स्थापनेचे विकल्प: स्थापना

By default, the local CD-ROMs are searched for the installation source. For a network install, select the स्थापना option. Possible installation protocols are

  • FTP
  • NFS
  • HTTP

The syntax to use is just like standard URLs. For example, if your server is found at 192.168.0.1 and you want to do an NFS-based install from the directory /install on this server, specify the source as follows:

  • स्थापना =एनएफएस://१९२.१६८.०.१/ स्थापना

The network card will either be configured with डिएचसिपि or you must specify the parameters yourself as described in the नेटवर्क विकल्प .
