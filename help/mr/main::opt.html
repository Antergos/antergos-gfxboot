बूट विकल्प

There are two types of boot options available. First, there are options that affect the installer. Second, there are kernel options. Some of the more common options are:

a) installer options

  • स्थापना -- select an installation source
  • नेटवर्क विकल्प -- the network options
  • विएनसि विकल्प -- options for installation via VNC

b) kernel options

  • स्प्लेश -- influence the behavior of the splash screen
  • एपिएम -- toggle power management
  • एसिपिआई -- advanced configuration and power interface
  • आईडिई -- control the IDE subsystem
