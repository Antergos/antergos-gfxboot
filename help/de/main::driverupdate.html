Treiberaktualisierung

Falls Sie eine Floppy oder CD-ROM zur Treiberaktualisierung benötigen, drücken Sie @@@fkey_dud@@@. Der Bootloader fordert Sie auf, nach dem Laden des Linux-Kernels das Treiberaktualisierungsmedium einzulegen.

Eine Treiberaktualisierung ist typischerweise eine Diskette mit neuen Versionen von Hardwaretreibern oder Fehlerbehebungen, die während der Installation benötigt werden.
